# Sistema Inteligente de Monitoramento de Saúde Mental para Estudantes

## Descrição

Este projeto propõe uma solução tecnológica para monitoramento da saúde mental de estudantes, utilizando coleta de dados comportamentais e inteligência artificial para identificar sinais de estresse, ansiedade e esgotamento.

---

## Problema

Estudantes enfrentam altos níveis de pressão acadêmica e não possuem acompanhamento contínuo de sua saúde mental.

As soluções atuais não são adaptadas ao ambiente educacional e não atuam de forma preventiva.

---

## Solução

O sistema propõe:

- Aplicativo mobile para registro de humor  
- Análise de dados com inteligência artificial  
- Sistema de alertas inteligentes  
- Dashboard para instituições de ensino  

---

## Tecnologias Utilizadas

- Backend: Node.js ou Python  
- Banco de dados: MongoDB  
- Mobile: React Native  
- API: REST  
- Inteligência Artificial: análise de padrões comportamentais
